Someone has added to your array without your permission. Luckily, they told you what they added.

What you need to do is remove from the array all the elements from the other array.

In the case of the same value being in your array twice, remove the last one.

Input: Two arrays.

Output: Array.

Example:

{% if interpreter.slug == "js-node" %} 
subtractArrays([0, 1, 2, 3, 4], [0, 2, 4]) == [1, 3]
subtractArrays([1, 2, 1], [1]) == [1, 2]
subtractArrays([0, [0, 1], "hello"], [[0, 1], "hello"]) == [0]
{% else %} 
subtract_arrays([0, 1, 2, 3, 4], [0, 2, 4]) == [1, 3]
subtract_arrays([1, 2, 1], [1]) == [1, 2]
subtract_arrays([0, [0, 1], "hello"], [[0, 1], "hello"]) == [0]
{% endif %}

Precondition: All elements in the second array are in the first
